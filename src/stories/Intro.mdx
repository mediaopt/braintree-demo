# Braintree Payment Demo

This project is a minimalistic demo environment for Braintree payment methods integrated with commercetools.

It covers the full pre-payment cart setup flow — product selection, shipping, discounts, customer assignment — and then initialises a Braintree checkout session to render the actual payment components.

## Setup

Our own demo environment is already provided. To test it wiht your own commercetools project, you can clone the repository and set up your own environment variables.

We strongly recommend to use for this demo the API client with commercetools checkout permissions only, to avoid any unwanted side effects on your commercetools project. CTP_SCOPE is not directly required for checkout itself - we use it to establish the cart before the checkout process.

Create a `.env` file in the project root with the following variables:

## Structure

We have two main demo modes in the project: checkout only loads the commercetools checkout module with predefined product and cart settings. Playground allwos to cover settings that should be dynamically obtained from server - like shipping modes available for different countries and preview the cart summary before starting the checkout process.
